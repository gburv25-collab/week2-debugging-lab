Week 2 Python Debugging Lab

This interactive lab supports Week 2: Debugging, Validation and Code Quality. Use it to run small Python examples, test inputs, identify defects, correct code and prepare Assignment 1 debugging evidence.

How to use this lab

Select an activity, read the task, run the code, test different inputs and record what you find. The aim is not just to fix the code, but to explain what was wrong and how the correction improves quality.

Some Python examples use input(). Because this page runs Python in the browser, you need to type the values for input() into the box labelled Program input for input() lines.

Use that box only when the code asks the user for input. If the program asks more than one question, put each answer on a new line.
